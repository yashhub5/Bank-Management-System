# 🏦 Flask Bank Management System

A secure Bank Management System built using Flask that provides user authentication and banking operations such as deposit, withdrawal, and balance inquiry using JWT-based authentication.

---

## 🚀 Features

- 🔐 User Authentication (JWT)
- 💰 Deposit Money
- 💸 Withdraw Money
- 📊 Check Account Balance
- 🏦 Banking APIs
- 🔒 Password Protected Routes
- ⚡ RESTful API Architecture
- 🗄️ PostgreSQL Database Integration
- 📝 ORM using SQLAlchemy
- ⚠️ Exception Handling
- 🌐 JSON-based API Responses
